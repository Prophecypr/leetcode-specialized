# Backtracking — 回溯

## 模板（C）

```c
void backtrack(参数列表) {
    if (满足结束条件) {
        记录结果;
        return;
    }
    for (int i = start; i < n; i++) {
        // 做选择
        path[pathSize++] = nums[i];
        // 剪枝（可选）
        if (需要剪枝) {
            pathSize--;
            continue;
        }
        // 递归
        backtrack(i + 1, ...);
        // 撤销选择
        pathSize--;
    }
}
```

## 常见类型

| 类型 | 特点 | 代表题 |
|------|------|--------|
| 子集 | 不需要剪枝，所有组合都要 | 78. 子集 |
| 全排列 | 需要标记已选元素 | 46. 全排列 |
| 组合总和 | 可重复选 → `backtrack(i, ...)` 不加 1 | 39. 组合总和 |
| N 皇后 | 按行逐行放置，检查列和对角线 | 51. N 皇后 |
| 单词搜索 | 二维 DFS，标记已访问 | 79. 单词搜索 |

## 易错点
- C 语言没有动态数组，需要预分配足够大的 `path[]` 和结果数组
- 做选择 → 递归 → 撤销选择，顺序不能错
- 剪枝条件：排序后跳过重复元素 `if (i > start && nums[i] == nums[i-1]) continue;`
